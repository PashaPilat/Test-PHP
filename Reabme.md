# 🎯 Задача проекта

## 🛒 Проект «Категории и товары»

> Учебный проект на **чистом PHP**, демонстрирующий реализацию каталога товаров  
> без использования фреймворков.

- [Функциональные требования](#функциональные-требования)
- [Дополнительные требования](#дополнительные-требования)
- [Дополнительная задача](#дополнительная-задача-2)
⬆ [Вернуться в главное меню](#-главное-меню)

---

## Функциональные требования

>1. Создать каталог товаров с категориями, связать их между собой.  
Для примера, можно взять данные с [demo.solomono.net](http://demo.solomono.net)  
>2. Создать в базе 10-20 товаров (товар должен иметь, дату, цену, назвавние товара), и 3-5 категорий
>3. Вывести страницу. 

Страница должна содержать:
- список категорий слева
- список товаров справа
- количество товаров в категории
- сортировку (по цене, алфавиту, новые), по дефолту дешовые
- AJAX обновление

>4.При клике на категорию, список товаров обновляется **без перезагрузки страницы**

У каждого товара:

- кнопка **Купить**
- открывается **модальное окно Bootstrap**

## Дополнительные требования

- ООП архитектура
- чистый PHP

Пример:


После обновления страницы состояние сохраняется.

## Builder

Билдер реализует **fluent interface**.

Пример:

```php
DB::select()
    ->from()
    ->where()
    ->orderBy()
    ->limit()
```

---

## Where

Логика фильтрации:

```
WHERE column operator value
```

---

## Join

Поддерживаются типы:

- INNER
- LEFT
- RIGHT
- FULL

---

## Enum для типизации

```
JoinType
JoinOperator
WhereOperator
Boolean
OrderDirection
```

Это позволяет избегать ошибок в SQL.

---

## 📖 Пример использования

```php
$rows = DB::select('id', 'name')
    ->from('products')
    ->where('status', 'active')
    ->orderBy('price', OrderDirection::ASC)
    ->all();
```

SQL:

```sql
SELECT id, name
FROM products
WHERE status = ?
ORDER BY price ASC
```

---

### JOIN пример

```php
$rows = DB::select('*')
    ->from('categories')
    ->join(
        'products',
        'categories.id',
        'products.category_id'
    )
    ->all();
```

SQL:

```sql
SELECT *
FROM categories
INNER JOIN products
ON categories.id = products.category_id
```
